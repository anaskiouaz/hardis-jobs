# Hardis Jobs

Plateforme de recrutement pour développeurs juniors, développée pour Hardis Group.

![HTML](https://img.shields.io/badge/HTML5-E34F26?style=flat&logo=html5&logoColor=white)
![CSS](https://img.shields.io/badge/CSS3-1572B6?style=flat&logo=css3&logoColor=white)

## 🎯 Fonctionnalités

- Interface de recrutement pour jeunes développeurs
- Formulaire de candidature
- Présentation des offres d'emploi
- Design moderne et responsive

## 📦 Installation

```bash
# Cloner le repository
git clone https://github.com/anaskiouaz/hardis-jobs.git

# Ouvrir dans un navigateur
cd hardis-jobs
# Ouvrir index.html dans votre navigateur
```

## 🛠️ Technologies

- **HTML5** - Structure
- **CSS3** - Styling et responsive design
- **JavaScript** - Interactivité (si applicable)
